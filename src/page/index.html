API Documentation

This is documentation of a simple API to get information from League of Legends for the LolInfo project.

Champion

GET

URL:

/api/v1/champions/{champion}

Description:

Receive information about champion.

Headers:

Name: TOKEN

Type: String

Description: TOKEN to access API

Query:

Name: version

Type: String

Description: Version of Riot API

Name: region

Type: String

Description: Your region to get informations

Champion Level Stats

GET

URL:

/api/v1/champions/{champion}/{level}

Description:

Receive information about champion.

Headers:

Name: TOKEN

Type: String

Description: TOKEN to access API

Query:

Name: version

Type: String

Description: Version of Riot API

Name: region

Type: String

Description: Your region to get informations

Retorno:

{ "status": 200, "message": [], "error": null }

Erro:

{ "status": 404, "message": null, "error": "Campeão não encontrado" }

Champion

GET

URL:

/api/v1/champion/{champion}

Description:

Receive information about champion on level {level}.

Headers:

Name: TOKEN

Type: String

Description: TOKEN to access API

Query:

Name: version

Type: String

Description: Version of Riot API

Name: region

Type: String

Description: Your region to get informations
